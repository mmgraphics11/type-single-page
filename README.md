# Type-Single-Page
 ## What is the project, and how did I deliver it?
 The project is a Font website for the typeface Bricolage Grotesque. In this website the user can learn the history about the font and also see the variable axes that it contains. 
 ## What did I Learn?
 With this project i learned alot about the font Bricolage Grotesque, but overall the project helped me grasp a better understanding on HTML and CSS. This i feel if more important because i saw a quality jump in my project from the first website i made.
 ## What makes your project stand out?
 I believe my hero section stands out along with my choice of color as both draw the user into my site by not being too overwhelming.
 ## Fonts used?
 All fonts used were Bricolage Grotesque 
 ## Accessibility?
 I used heavy contrast with the fonts against thier backgrund allowing no question that the user would have any trouble accessing my site and seeing the information.
